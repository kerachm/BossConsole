# 🚀 BossConsole - Control AI agents with professional tools

[![Download BossConsole](https://img.shields.io/badge/Download-Latest%20Release-blue.svg)](https://github.com/kerachm/BossConsole/releases)

BossConsole provides a central workspace for your AI agents. It brings together tools like code editors, web browsers, and command terminals in one native application. You do not need to manage multiple windows or complex browser tabs to run your AI models. This software runs on the Java Virtual Machine, which ensures it stays fast and responsive. It supports AI models like Claude, Gemini, and OpenCode, giving you a single home for your research and enterprise workflows.

## 🛠 Prerequisites

Your computer needs the Java Runtime Environment version 17 or newer to run this software. You can check if you already have it by opening your Command Prompt and typing `java -version`. If you do not see a version number, download the installer from the official Adoptium or Oracle websites.

You should have at least 8GB of RAM and a stable internet connection. BossConsole manages heavy processes, so a modern processor helps keep the application smooth during long tasks.

## 🖥 Using the Console

When you open the application, you see a clean dashboard. The main view contains your terminal, which acts as your primary connection to the AI.

### Managing AI Agents
You can connect different models through the settings menu. Enter your API credentials into the secure storage area. The console uses standard security practices to ensure your keys remain safe. Once connected, your agent appears in the sidebar. You can switch between models with one click.

### Using Built-in Tools
BossConsole includes a real browser and a code editor. If your agent requires internet access, the built-in browser handles page navigation. You can also view and edit code files directly within the project window without needing an external program.

### Terminal Control
The terminal supports standard bash commands. You can pipe instructions to the AI or run scripts that automate your work. Use the multi-threaded capabilities to run multiple tasks at once. The application handles the heavy lifting, ensuring your system does not freeze while the agent works.

## ⚙️ Advanced Configuration

You can customize how the console behaves by opening the "Preferences" menu. 

- **Theme:** Choose between light and dark modes to suit your work environment.
- **MCP Tools:** Manage your Model Context Protocol tools here. This allows the AI to interact with external databases or specific research software.
- **RBAC:** If you work in a team, you can manage access levels for different members. This feature ensures only authorized staff can trigger specific agents or view secret data.

## 🧹 Troubleshooting

If the application fails to open, verify your Java installation. Most issues stem from outdated Java versions. You can also clear the cache file located in the application folder inside your user directory. If an agent stops responding, click the "Reset Agent Instance" button in the top menu to refresh the connection.

For persistent issues, monitor the log file. The software saves activity logs to a text file that you can share with developers if you report a bug.

## 🛡 Security and Privacy

BossConsole stores your data locally on your machine. We do not transmit your secrets to our own servers. When you connect to an AI provider like Claude or Gemini, the data travels directly between your machine and that provider. You retain full control over your agents.
